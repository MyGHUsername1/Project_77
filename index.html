All About Visual Studio Code

This is an explanation about Visual Studio Core, which is a platform for code. You can use many different coding languages within this platform, such as Python, HTML, and Javascript.

    The following will be step-by-step instructions on how to create an HTML page on a Mac, and how to add Javascript and CSS.


  • Download and open Visual Studio Core.
  • Download Live Server extension in VSC
  • Create new file: Click menu File -> New File
  • Save created file: Click menu File -> Save As, named index.html in a folder on desktop.
  • This file/folder can be found using Finder.
  • Code HTML requrirements in index.html
  • Repeat Step 3 and 'Save As' main.js in the same folder as index.html.

  • Code main.js to have functions, queries, as per requirements.
  • Repeat Step 3 and 'Save As' style.css in the same folder as index.html.
  • In .HTML file, code: '' to link Stylesheets from style.css
  • Code designs and formattings in style.css as per requirements.
  • In the bottom right corner of VSC, click on the 'Go Live' button.

  • To view Console output, control+click, then click Inspect.
  • If there are errors during Runtime, it will show in Console.
